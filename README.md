# WTFGPL
Do What The Fuck You Want To Greater Public License, a modified version of <[WTFPL](https://www.wtfpl.net/about/)>, with "AS IS" clause and "Don't bitch to me" condition.  
![alt text](https://github.com/DS1TPT/wtfgpl/blob/main/wtfgpl.png)  
What is this?
-------------- 
WTFGPL is a very permissive license that allows any user to do whatever the user wants, just like the original <[WTFPL](https://www.wtfpl.net/about/)>.
This license is an "as is" license by default, which means the work comes without any warranty and the creator shall not be held liable for claims and other bullshit.  

I've seen some crazy guys annoying the hell out of the creator, even though they didn't pay for or contribute to it.  
This license mandates the user to not bitch to the creator in order to use the work, so that we have one less thing to worry about.  
The only thing you have to do is to upload your great work and forget about it. If someone bitches to you about their garbage anyways, you just say, "Well, read the damn license and fuck off".  

How do I use the WTFGPL?
-------------------------
Just put this copyright and license statements to your beautiful piece of art:
```
Copyright (C) [YEAR] [YOUR NAME] <your@address.com>

This work is provided "AS IS". You can redistribute it and/or modify it 
under the terms of the Do What The Fuck You Want To Greater Public
License, as published by Lee Geon-goo <github.com/DS1TPT>. You should
have received a copy of the Do What The Fuck You Want To Greater Public
License. If not, see <https://github.com/DS1TPT/wtfgpl> for more details.
```
You may use the following statements below, if you don't want to waste space:
```
Copyright (C) [YEAR] [YOUR NAME] <your@address.com>
This work is provided "AS IS" under the terms of the Do What The Fuck You
Want To Greater Public License. See LICENSE or <https://github.com/DS1TPT/wtfgpl>.
```
After you put the copyright statements, be sure to include the LICENSE file(or printed version if you really want to use this for something other than software) to the work.  
If for some reasons you cannot provide LICENSE file(or printed matter) separately, you can just use the full license text instead of the statements above, as it has separate copyright statement for you to write the year and your name on.
